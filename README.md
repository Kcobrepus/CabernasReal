Cabernas 

Cabernas é um joga de plataformas onde o jogador controla um personagem (Bernas) com as setas do tecaldo ou as teclas A e D para se mover para a esquerda e direita, e clicando no espaço para saltar.

O objtivo de Cabernas é levar o Bernas até à escada que o leva para fora da caverna, tendo de desviar de inimigos e armadilhas para conseguir omorango que lhe concede o poder do duplo salto.
